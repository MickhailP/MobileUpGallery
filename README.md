
## Приложение для просмотра фотографий из альбомов VK

### Технологии:
- UIKit
- MVVM + Coordinator
- VK OAuth 2.0
- Async/await
- Code layout
- Target iOS 15.0, Target Device- iPhone
- Кэширование изображений
- Поддержка Dark/Light mode 
- Keychain
- GitFlow
- Eng/Rus localization 
- Custom reusable views 

SPM:
- SnapKit
